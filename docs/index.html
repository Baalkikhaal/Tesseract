Tesseract - A project on optical character recognition

This page documents the Tesseract project. Tesseract is library that provides optical character recognition functionality using neural networks. We use PyTesseract, a Python wrapper for Tesseract library to access the library's routines. The library was tested on scanned PDF copy of the famous book Weaving the Web. Since tesseract library takes in png images as input, I need to first the convert the PDF pages into png. Poppler is a linux library that handles pdf to png conversion. However to use Poppler in window, one needs Windows Subsystem for Linux. The details of installing and using Poppler in Windows are found in the README of the project. I wrote a small Python script "pdftopng.py" to convert pdf to png. And another small Python script "pngtotxt.py" that processes a png image to text To have a look at the source code and example Python script check the Tesseract repository
